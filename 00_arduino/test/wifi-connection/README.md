# setup
- mac local server
  - $ `python -m SimpleHTTPServer`
- to check address
  - $ `ifconfig`
- add wifi-info.h
  ```
  #define WIFI_SSID "xxxx"
  #define WIFI_PASSWORD "xxxx"
  #define SERVER_URL "http://xxx.xxx.xxx.xxx:8000/"
  ```
